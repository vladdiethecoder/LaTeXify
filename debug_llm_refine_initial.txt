\section*{Question 3}
(6 points)

\textbf{a)}
\begin{itemize}
    \item Inverse (reflection across \( y = x \)).
    \item Vertical line test: fails.
\end{itemize}
Construct inverse: reflect graph of \( y = p(x) \) across \( y = x \)
\[
\Rightarrow \text{graph of } p^{-1}
\]
Apply VLT to \( p^{-1} \): some verticals cut the graph twice
\[
\Rightarrow \text{Inverse is not a function}
\]
Domain/Range swap: \(\text{Dom}(p^{-1}) = \text{Ran}(p)\), \(\text{Ran}(p^{-1}) = \text{Dom}(p)\)

\textbf{b)}
\begin{itemize}
    \item Inverse of a line (reflection across \( y = x \)).
    \item Vertical line test: passes.
\end{itemize}
Construct inverse: reflect non-vertical line across \( y = x \)
\[
\Rightarrow \text{another non-vertical line}
\]
Apply VLT to \( f^{-1} \): each vertical meets at most once
\[
\Rightarrow \text{Inverse is a function}
\]
Domain/Range swap: \(\text{Dom}(f^{-1}) = \text{Ran}(f)\), \(\text{Ran}(f^{-1}) = \text{Dom}(f)\)

\section*{Question 5}
(8 points)

The point (1, −2) is on the graph of \( f \). Describe the following transformations on \( f \), and determine the resulting point.

We use
\[
g(x) = a f(k(x - d)) + c,
\]
\[
x' = \frac{x}{k} + d,
\]
\[
y' = a y + c.
\]

\textbf{a)} \( g(x) = 2f(x) + 3 \)

The \( a = 2 \) indicates a vertical stretch by a factor of 2 and the \( c = 3 \) indicates a vertical translation of 3 units up.
\[
x' = \frac{x}{k} + d = 1 \cdot \frac{1}{1} + 0 = 1
\]
\[
y' = a y + c = 2(-2) + 3 = -1
\]
Therefore, the resulting point is (1, −1).

\textbf{b)} \( g(x) = f(x + 1) - 3 \)

The \( d = -1 \) (since \( x - d = x - (-1) = x + 1 \)) indicates a horizontal translation of 1 unit to the left and the \( c = -3 \) indicates a vertical translation of 3 units down.
\[
x' = \frac{x}{k} + d = 1 \cdot \frac{1}{1} + (-1) = 0
\]
\[
y' = a y + c = 1(-2) + (-3) = -5
\]
Therefore, the resulting point is (0, −5).

\textbf{c)} \( g(x) = -f(2x) \)

The \( a = -1 \) indicates a reflection in the x-axis and the \( k = 2 \) indicates a horizontal compression by a factor of \( \frac{1}{2} \).
\[
x' = \frac{x}{k} + d = 1 \cdot \frac{1}{2} + 0 = \frac{1}{2}
\]
\[
y' = a y + c = (-1)(-2) + 0 = 2
\]
Therefore, the resulting point is \( \left( \frac{1}{2}, 2 \right) \).

\textbf{d)} \( g(x) = -f(-x - 1) + 3 \)

The \( a = -1 \) indicates a reflection in the x-axis, the \( k = -1 \) indicates a reflection in the y-axis, the \( d = -1 \) (from \( x - d = x - (-1) = x + 1 \)) indicates a horizontal translation of 1 unit to the left, and the \( c = 3 \) indicates a vertical translation of 3 units up.
\[
x' = \frac{x}{k} + d = 1 \cdot \frac{1}{-1} + (-1) = -2
\]
\[
y' = a y + c = (-1)(-2) + 3 = 5
\]
Therefore, the resulting point is (−2, 5).
